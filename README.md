# Basic Python Programs

This repository contains two simple yet essential Python programs that demonstrate fundamental programming concepts: basic mathematical operations and personalized user greetings.

## Features

- **Task 1: Mathematical Operations Calculator**
  - User inputs two numbers
  - Performs addition, subtraction, multiplication, and division
  - Displays all operation results clearly

- **Task 2: Personalized Greeting Generator**
  - Takes first name and last name as input
  - Concatenates names into full name
  - Generates customized greeting message

## Task 1 - Mathematical Operations

**`Task1.py`** performs basic arithmetic operations on two user-provided numbers.

### Functionality
```
1. Input: Two numbers (float support)
2. Operations: +, -, *, /
3. Output: Results of all four operations
```

**Sample Output:**
```
Enter first number: 10
Enter second number: 5
Addition: 15.0
Subtraction: 5.0
Multiplication: 50.0
Division: 2.0
```

## Task 2 - Personalized Greeting

**`Task2.py`** creates a customized greeting using user's full name.

### Functionality
```
1. Input: First name and last name
2. Processing: Concatenate names
3. Output: Personalized greeting message
```

**Sample Output:**
```
Enter your first name: G Meher
Enter your last name: Prateek
Hello, G Meher Prateek! Welcome to Python programming!
```

## How to Run

1. **Clone the repository:**
   ```bash

   ```

3. **Run Task 2:**
